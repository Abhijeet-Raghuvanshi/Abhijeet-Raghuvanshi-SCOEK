# Seva Industries WebView App

This Android app wraps the Seva Industries website in a modern Material UI with bottom navigation for Home, Products, and Contact.

## How to open and run

1. Open Android Studio.
2. Choose "Open" and select this folder (`D:\Sample app`).
3. Let Gradle sync finish.
4. Connect an Android device (USB with USB debugging) or start an emulator.
5. Click Run.

## Features
- WebView with JavaScript and DOM storage enabled
- Pull-to-refresh and loading progress bar
- Bottom navigation:
  - Home → `https://sevaindustries.in/`
  - Products → `https://sevaindustries.in/products/`
  - Contact → `https://sevaindustries.in/contact/`

## Notes
- Requires Internet permission (declared in manifest).
- Icons use system placeholders; replace with brand assets as needed.
